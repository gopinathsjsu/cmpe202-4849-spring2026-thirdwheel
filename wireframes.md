# Zestify ‚Äî UI Wireframes

Low-fidelity wireframes for every screen + email template in the app. All PNGs
live in [`wireframes/`](wireframes/).

| # | Screen | Wireframe |
|---|--------|-----------|
| 1 | **Landing page** `/` ‚Äî hero, featured events, categories, "Why Zestify" highlights. | [`1.png`](wireframes/1.png) |
| 2 | **Events listing** `/events` ‚Äî search + filter row, 3-column responsive grid, pagination. | [`2.png`](wireframes/2.png) |
| 3a | **Event detail ‚Äî attendee** `/events/[id]` ‚Äî banner, schedule, map embed, "Register / Get Tickets" sidebar. | [`3a.png`](wireframes/3a.png) |
| 3b/3c | **Event detail ‚Äî organizer + admin** ‚Äî same page swaps sidebar for Reschedule / Cancel / View Attendees actions. | [`3b3c.png`](wireframes/3b3c.png) |
| 4 | **Reschedule modal** ‚Äî date/time pickers + reason field, fires email to all attendees. | [`4.png`](wireframes/4.png) |
| 5 | **Ticket purchase modal** ‚Äî quantity, total, Stripe Elements `<PaymentElement>` for paid events. | [`5.png`](wireframes/5.png) |
| 6/7 | **Login + register** `/login` `/register` ‚Äî auth form + 3 quick-login buttons (Admin / Organizer / Attendee). | [`6-7.png`](wireframes/6-7.png) |
| 8 | **My tickets** `/dashboard/my-tickets` ‚Äî ticket cards with code + cancel button per row. | [`8.png`](wireframes/8.png) |
| 9 | **My events (organizer)** `/dashboard/my-events` ‚Äî 4-stat band + event cards with Edit / Cancel / Reschedule. | [`9.png`](wireframes/9.png) |
| 10 | **Attendees** `/dashboard/attendees/[id]` ‚Äî per-event attendee table with CSV export. | [`10.png`](wireframes/10.png) |
| 11 | **Admin dashboard** `/admin` ‚Äî 4 stat cards + tabs (Pending / All / Users / Audit Log) + moderation queue. | [`11.png`](wireframes/11.png) |
| 12 | **Notifications** `/notifications` ‚Äî unread/read list of system notifications, mark-all-read. | [`12.png`](wireframes/12.png) |
| 13 | **Create event** `/events/create` ‚Äî full event form with Google Maps link parser + lat/lon autofill. | [`13.png`](wireframes/13.png) |
| 14 | **Email templates** ‚Äî confirm / cancel / reschedule / 12h-reminder / ticket-cancel; all carry the cyan "Open in Google Maps" button. | [`14.png`](wireframes/14.png) |
